# AI Camp - Session 4: Natural Language Processing

This folder contains the materials for the fourth AI Camp session. The session starts with classical Natural Language Processing techniques and progresses to modern Transformer architectures and BERT-based Named Entity Recognition.

## Topics

- Text preprocessing: lowercasing, punctuation removal, stopword removal, stemming, and lemmatization
- Linguistic analysis: lexical, morphological, syntactic, and semantic layers
- Tokenization and Byte-Pair Encoding
- Text representation with Bag-of-Words, N-grams, and TF-IDF
- Word2Vec, GloVe, and contextual BERT embeddings
- Sequence models: RNNs, LSTMs, and GRUs
- Attention, self-attention, and Transformer architecture
- BERT and GPT architecture, training objectives, and text generation
- Named Entity Recognition using BERT token classification

## Contents

| File | Description |
| --- | --- |
| `notebook/NLP_Session4_Classical_to_Transformers.ipynb` | Main session notebook covering classical NLP, embeddings, sequence models, attention, BERT, and GPT. |
| `NER.ipynb` | Practical BERT-based Named Entity Recognition training and evaluation notebook. |
| `resources.txt` | Recommended courses, videos, and original research papers for further study. |

## Recommended Flow

1. Start with `notebook/NLP_Session4_Classical_to_Transformers.ipynb`.
2. Review the progression from text preprocessing to Transformer models.
3. Open `NER.ipynb` to apply BERT to a token-classification task.
4. Use `resources.txt` for additional study.

The notebooks use libraries including NLTK, NumPy, pandas, Matplotlib, PyTorch, Transformers, Sentence Transformers, and scikit-learn. They can be opened locally with Jupyter Notebook or uploaded to Google Colab.
